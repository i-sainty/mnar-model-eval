https://www.kaggle.com/datasets/nancyalaswad90/finding-donors-for-charityml
This is a modified version of the classic UCI Census dataset, specifically framed around predicting individuals who make more than $50K to target for charity mailings. It’s perfect for simulating MNAR. For instance, you could design the experiment so that individuals in certain lower-paying job classes have a higher rate of missing financial data, directly testing how MICE and XGBoost handle that structural degradation.
